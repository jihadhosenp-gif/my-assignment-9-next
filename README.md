#  StudyNook - Study Room Booking System

A modern full-stack study room booking platform built with **Next.js, React, Node.js, Express, and MongoDB**.  
Users can browse rooms, view details, and book study spaces easily.

---

##  Features

-  Browse study rooms (cards UI)
-  View room details
-  Book study rooms
-  Favorite rooms system (UI ready)
-  Authentication (Better Auth)
-  MongoDB database integration
-  Fully responsive UI
-  Modern SaaS-style design

---

##  Tech Stack

### Frontend:
- Next.js (App Router)
- React.js
- Tailwind CSS
- React Icons

### Backend:
- Node.js
- Express.js
- MongoDB (Native Driver)
- Better Auth
